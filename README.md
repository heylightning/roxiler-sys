### Run the main.js file:

1. Clone the repo:

```
git clone <repo-link>
```

2. Change directory to the repo:

```
cd <repo-name>
```

3. Install the dependencies:

```
npm install
```

4. Run the main.js file:

```
node main.js
```

### Tech stack used.

1. NodeJS
2. ExpressJS
